h1. ClearCache Plugin

Source code of the intellij IDEA "ClearCache plugin": http://plugins.intellij.net/plugin/index?pr=&pluginId=7197
This is a plugin for the various intellij IDEs, working in IDEA, PhpStorm, AppCode, etc.



h2. Please note

This plugin is distributed on an "AS IS" BASIS, WITHOUT WARRANTIES OR CONDITIONS OF ANY KIND, either express or implied.



h2. Description

The plugin adds a "Clear Cache" button to the main toolbar: when clicked it will recursively delete all files/folders inside the cache folder(s).
The cache folder(s) are configured in the project settings.
